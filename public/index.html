Local Conversion Console

把图片、文档、PDF 和媒体转换，收拢到一个工作台里。

DocTranslater 面向个人使用场景：浏览器负责发起任务，真正的文件处理由你自己的机器或服务器完成。

进入工作台

适合部署在自己的电脑、内网机器或带 HTTPS 的个人服务器上。

Notes

适合的使用方式

  • 图片、PDF、文档、音视频使用统一入口，避免工具分散。
  • 大文件任务优先看环境状态和执行日志，再判断是依赖问题还是格式问题。
  • 如果部署到公网，优先保持 HTTPS、Nginx 上传上限和系统依赖一致。

Workspace

转换工作台

每张卡片都是一个独立工作流：上传、选择目标格式、提交任务，然后直接下载结果。

01

Image

图片格式互转

在常见图片格式之间切换，适合压缩、兼容性处理和网页素材准备。

输入：PNG / JPG / WEBP

02

Document

文档转 PDF

把 Word 文档整理成更稳定的 PDF 版本，适合共享、归档和打印。

输入：DOCX | 输出：PDF

03

PDF

PDF 拆分

按页拆成多个 PDF，并自动打包下载，适合材料拆页和逐页分发。

输出：ZIP 包

04

PDF

PDF 合并

按你选择的顺序把多个 PDF 合成一份，适合资料汇总和提交版本整理。

输入：至少 2 个 PDF

05

PDF

PDF 转图片

按页导出成图片，适合做预览图、单页分享和内容摘取。

输出：PNG / JPG + ZIP

06

Text

Markdown / HTML

在纯文本和网页结构之间双向转换，适合文档整理和内容迁移。

支持双向互转

07

Media

音视频转码

针对常见的音视频素材做基础转码，快速得到更适合分发的格式。

输入：MP3 / MP4 / MOV / MKV / WAV / M4A

Console

执行日志

每次请求、错误和下载结果都会记在这里，排查问题时先看这一块。
